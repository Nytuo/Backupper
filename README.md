## About

Backupper is a lightweight and user-friendly backup application that makes
protecting your important files effortless. Built with modern web technologies
and Tauri, it provides a cross-platform solution for reliable file backups.

## Features

- **Simple backup management** — Create, perform and manage backups with an
  intuitive interface
- **Cross-platform support** — Works seamlessly on Windows, macOS, and Linux
- **Fast and efficient** — Optimized performance with minimal resource usage
- **Secure storage** — Protect your backups with reliable storage solutions
